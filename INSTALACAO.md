# Como instalar o Design System da EnP no Cursor

**Tempo:** 30 segundos

---

## O que é

Uma skill que ensina a IA do Cursor a seguir as regras visuais da EnP automaticamente — cores, fontes, botões, espaçamentos, tudo no padrão da marca.

---

## Como instalar

1. Abra o **Cursor**
2. Abra o chat da IA: **Ctrl + L**
3. Cole a mensagem abaixo e envie:

```
Instale a skill enp-design-system a partir do repositório https://github.com/eduardogasparetto-cpu/enp-design-system — clone o repositório e copie os arquivos: skill/SKILL.md para ~/.cursor/skills/enp-design-system/SKILL.md e rule/enp-design-system.mdc para ~/.cursor/rules/enp-design-system.mdc. Crie as pastas se não existirem.
```

4. Reinicie o Cursor.

---

## Como testar

No chat do Cursor, mande:

```
Crie um botão primário seguindo o design system da EnP
```

Se a IA usar o azul #1076DD e a fonte Plus Jakarta Sans, está funcionando.

---

## Como atualizar no futuro

No chat do Cursor, mande:

```
Atualize a skill enp-design-system — faça git pull no repositório https://github.com/eduardogasparetto-cpu/enp-design-system e copie os arquivos atualizados para ~/.cursor/skills/ e ~/.cursor/rules/
```

---

## Se der erro

Cole a mensagem de erro no chat — a IA corrige sozinha.
